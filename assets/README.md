# assets/

Drop project images here.

## Logos (used by the About section on the landing page)

Add the two logos you have from the poster:

- `iti-logo.svg` (or `.png`) — ITI logo
- `sphinx-logo.svg` (or `.png`) — Sphinx logo

Then, in `index.html`, replace the text placeholders inside
`.about-logos` with real images:

```html
<img class="about-logo-img" src="assets/iti-logo.svg" alt="ITI" />
<img class="about-logo-img" src="assets/sphinx-logo.svg" alt="Sphinx" />
```

## Screenshots (optional, for the README)

Add `screenshots/workspace.png`, `screenshots/catalog.png`, etc. and reference
them from `README.md`.
